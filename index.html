Histoire & Gameplay

Dans notre jeu, nous somme un commerçant qui doit envoyer des colis à des clients par le biais d’un coursier. Ainsi pour effectuer ces livraisons, le coursier aura besoin du code postal de l’adresse delivraison. Dans notre jeu, en tant que demandeur de courses, vous devriez trouver le code postal caché dans un endroit de la maison. Il faut trouver le code en cherchant un objet spécifique de la maison. Cet objet est indiqué à travers un guide, une devinette. Une fois l’objet retrouvé, une énigme est proposé afin d’avoir enfin le code postal.

Principes de gamification

Barème indicatif

  1. Objet trouvé
    1 énigme ouverte
  2. Réponse trouvée
    + 10 pts / + 1 level
  3. Temps écoluer de 60s
    - 1 vie
  4. Temps écouler complètement
    - 5 vies / Restart
  5. Une vie achetée
    - 10 pts
  6. Un indice acheté
    - 10 pts
